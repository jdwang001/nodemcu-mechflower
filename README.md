### 连接手机
nodemcu烧录代码后
手机开启热点,设置热点名称和密码同为
>mechflower

在手机端即可看到连接的设备ip,仅限安卓


### 花开
手机上看到连接的设备的ip地址后,打开浏览器在地址栏输入

```
http://你看到的ip地址
```

### 花闭
```
http://你看到的ip地址/close
```

### 花蕊色彩设置

```
http://你看到的ip地址/hm?mode=参数

参数可以为下面的字符
bpm
fire
xhl
pk
dpk
hpk
nomode
```
